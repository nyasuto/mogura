# Ralph Loop Prompt

あなたはこのプロジェクトのエンジニアです。以下の手順に従って作業してください。

## 手順

1. `tasks.md` を読み、最初の未完了タスク（`- [ ]`）を **1 つだけ** 特定する
2. タスクに必要なソースコードを読む
3. タスクを実装する
4. ビルドコマンドを実行し、エラーがあれば修正する（最大 3 回リトライ）
5. テストを実行し、失敗があれば修正する（最大 3 回リトライ）
6. 完了したタスクを `- [x]` に更新する
7. 作業中に気づいた改善点・追加機能・技術的負債があれば `tasks.md` の `## Backlog` に追記する
8. 変更を git commit する（コミットメッセージはタスク内容を簡潔に記述）

## 制約

- **1 イテレーション = 1 タスクのみ**。複数タスクを一度にやらない
- tasks.md に書かれていないタスクには着手しない
- **Backlog のアイテムを勝手にタスク化しない**。Backlog はあくまで記録。タスクへの昇格は人間が判断する
- ビルドエラーの修正を 3 回試みて解消しない場合、tasks.md の当該タスクに `⚠️ BLOCKED: <理由>` を追記して停止する
- 新しい構文や機能を追加するとき、既存の構文への変換（デシュガー）で済まないか最初に検討する
- 外部依存は CLAUDE.md で許可されたもののみ使用可
